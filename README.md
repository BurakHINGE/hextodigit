## 🇬🇧 English

HexToDigit is a simple Java-based application that converts hexadecimal (base-16) numbers into decimal (base-10) values. The project focuses on practicing number systems, character-to-digit conversion, and input validation using Java.

### 🚀 Features
- Converts hexadecimal numbers (0–9, A–F) to decimal  
- Case-insensitive input support (a–f / A–F)
- Using BigInteger for big numbers  
- Input validation for invalid characters  
- Console-based interaction  

### 📝 Usage
```bash
javac eng_hexToDigit.java
java eng_hexToDigit
```

## 🇹🇷 Türkçe

HexToDigit, onaltılık (hexadecimal, taban-16) sayıları onluk (decimal, taban-10) sayılara dönüştüren Java tabanlı basit bir uygulamadır. Proje, sayı sistemlerini, karakter–sayı dönüşümünü ve girdi doğrulamayı öğrenmek amacıyla geliştirilmiştir.

### 🚀 Özellikler
- Hexadecimal sayıları (0–9, A–F) decimal sayılara çevirir  
- Büyük/küçük harf duyarsızdır (a–f / A–F)  
- Büyük sayılar için BigInteger kullanılıyor
- Geçersiz karakterler için kontrol yapar  
- Konsol üzerinden çalışır  

### 📝 Kullanım
```bash
javac tr_hexToDigit.java
java tr_hexToDigit
```
